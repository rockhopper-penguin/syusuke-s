# MySite

## 言語・フレームワーク

HTML5 + CSS3 + JavaScript

## ホスティング

Firebase Hosting

---

# デプロイ

1. Firebase cli のインストール(インストール済みの場合はスキップ)

```bash
npm install -g firebase-tools
```

2. Firebase にログイン

```bash
firebase login
```

3. `firebase.json`が存在するフォルダで以下コマンドを実行して Firebase Hosting にデプロイ

```
firebase deploy
```
